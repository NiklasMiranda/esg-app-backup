Du kan ændre farverne ved at redigere props'ene `color1`, `color2` og `color3`, der sendes til `Grainient`-komponenten i `src/components/LandingPage.js`.

For eksempel kan du ændre dem således:

```javascript
          <Grainient
            color1="#FF0000" // Rød
            color2="#00FF00" // Grøn
            color3="#0000FF" // Blå
            // ...andre props
          />
```

Husk at gemme filen efter dine ændringer for at se effekten.
